Frontend Developer (Intermediate)

We’re looking for an intermediate frontend developer to help us build useful, intuitive, open-source websites and tools.

Our projects range from small, single-page embeddable tools to large websites with rich content. The succesful applicant will help design and implement user-focused projects, with a particular emphasis on understanding and supporting user needs. This is an opportunity to dig deep to ensure users get the most out of what we build.

Candidates should be well-versed in modern frontend development tools and best-practices, be strong advocates for a good user experience, and be able to work well independently and collaboratively.

We're particularly interested in candidates that share our passion for working to help South Africans make more informed decisions.

The position is full-time and based in Cape Town.

Requirements

  • At least 3 years' experience building web applications and frontends
  • Solid knowledge of Javascript, jQuery, CSS3 and HTML5
  • Solid knowledge of at least some of the modern frontend technologies, such as Angular, Backbone, React, Handlebars, Underscore, D3, Vue, etc.
  • Experience implementing frontends using a REST API
  • Experience using Agile processes in a small team
  • Familiarity with GitHub and GitHub Pages
  • Has their own laptop
  • Non-South Africans must have a valid work permit

How to apply: You can apply for a position by sending an email to jobs@openup.org.za. Please attach both your CV and a cover letter.
