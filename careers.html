Frontend Developer (Intermediate)

We’re looking for an intermediate, experience frontend developer to help us build useful, intuitive, open-source websites and tools.

Our projects range from small, single-page embeddable tools to large websites with rich content. The succesful applicant will help design and implement these projects and ensure they're helpful and user-friendly.

Candidates should be well-versed in modern frontend development tools and best-practices, be strong advocates for a good user experience, and able to work well independently and collaboratively.

We're particularly interested in candidates that share our passion for working to help South Africans make more informed decisions.

The position is based in Cape Town.

Requirements

  • At least 3 years' experience building web applications and frontends
  • Solid knowledge of Javascript, jQuery, CSS3 and HTML5
  • Solid knowledge of at least some of the modern frontend technologies, such as Angular, Backbone, React, Handlebars, Underscore, D3, Vue, etc.
  • Experince implementing frontends using a REST API
  • Experince using Agile processes in a small team
  • Familiarity with GitHub and GitHub Pages
  • Has their own laptop
  • Permission to work full-time in South Africa

How to apply: You can apply for a position by sending an email to jobs@openup.org.za. Please attach both your CV and a cover letter.
