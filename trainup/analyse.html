Open data initiatives and data literacy have become increasingly important as the digital age opens up new ways for people to access and interrogate information. This brings power to accountability, where data is the common language and will disclose any information that is hidden. As a result, data storytelling plays a pivotal role.

We need to understand how to analyse this data in order to discover new issues, gain a better understanding of the reality, and to be able to verify facts from sources.

This course provides participants with a blueprint to follow in order to interview a dataset, analyse the data, and discover the stories hidden in the data. A spreadsheet forms the knowledge base that provides the springboard for those wishing to tell stories using data into the world of data analysis.

Objectives

  • Establish a set of principles and guidelines to consider when planning and producing a data driven story
  • Perform comprehensive spreadsheet functionality
  • Analyse and identify trends and patterns in data
  • Find new angles hidden within datasets
  • Compose basic overviews of data utilising pivot tables
  • Combine different sources of data in order to enrich a dataset
  • Manage small to medium datasets

Modules

  • Philosophy of Data Storytelling
  • Data Analysis in Perspective
  • Fundamentals of Data Analysis
  • Formulae & Functions
  • Ethics for Data Analysis

Prerequisites

  • What is data
  • Data Pipeline
  • Importing CSV data
  • Data Unit
